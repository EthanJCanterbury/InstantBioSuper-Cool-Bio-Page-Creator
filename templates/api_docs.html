API Documentation

Bio Page Creator provides a RESTful API that allows you to programmatically create, read, update, and delete bio pages.

Base URL

All API URLs referenced in this documentation are relative to your deployment URL.

Authentication

Currently, the API does not require authentication. This may change in future updates.

Response Format

All responses are returned in JSON format. Each response includes a success boolean that indicates whether the request was successful.

Endpoints

GET /api/bios

Returns a list of all available bios.

Response

{
  "success": true,
  "count": 2,
  "bios": ["john_doe", "jane_smith"]
}

GET /api/bio/{bio_name}

Returns details for a specific bio.

Response

{
  "success": true,
  "bio_name": "john_doe",
  "bio": {
    "name": "John Doe",
    "tagline": "Web Developer",
    "bio": "Hi, I'm John. I love creating web applications and solving problems.",
    "profile_image": "/uploads/1234abcd.jpg",
    "theme": "dark",
    "social_links": {
      "twitter": "johndoe",
      "instagram": "johndoe",
      "linkedin": "johndoe",
      "github": "johndoe"
    },
    "custom_css": ""
  }
}

POST /api/bio/{bio_name}

Creates a new bio with the specified name.

Request
Response

Request Body

{
  "name": "John Doe",
  "tagline": "Web Developer",
  "bio": "Hi, I'm John. I love creating web applications and solving problems.",
  "profile_image": "/uploads/1234abcd.jpg",
  "theme": "dark",
  "social_links": {
    "twitter": "johndoe",
    "instagram": "johndoe",
    "linkedin": "johndoe",
    "github": "johndoe"
  },
  "custom_css": ""
}

Response

{
  "success": true,
  "message": "Bio created successfully",
  "bio_name": "john_doe"
}

PUT /api/bio/{bio_name}

Updates an existing bio with the specified name.

Request
Response

Request Body

{
  "tagline": "Full Stack Developer",
  "bio": "Updated bio information here"
}

Response

{
  "success": true,
  "message": "Bio updated successfully",
  "bio_name": "john_doe"
}

DELETE /api/bio/{bio_name}

Deletes a bio with the specified name.

Response

{
  "success": true,
  "message": "Bio deleted successfully",
  "bio_name": "john_doe"
}

GET /api/themes

Returns a list of available themes for bios.

Response

{
  "success": true,
  "count": 5,
  "themes": ["default", "dark", "light", "colorful", "minimal"]
}
